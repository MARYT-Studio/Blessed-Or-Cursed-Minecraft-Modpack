# Blessed-Or-Cursed-Minecraft-Modpack

Repo for all resources of *Blessed or Cursed* Minecraft Modpack.

Design of this modpack's game window icon, or both `icon16x16.png` and `icon32x32.png` namely, is from https://game-icons.net/, licensed under CC BY 3.0. Great thanks to the original designer.

This integration package includes modified versions of the GUI textures from the Heat And Climate Lib (all located in the oresources\dcs_climate\textures\gui directory); Original files [released under the CC-BY-NC-SA 4.0 license](https://github.com/defeatedcrow/HeatAndClimateLib?tab=readme-ov-file#assets-cc-by-nc-sa-40-). We hereby express our gratitude to the original author.
